## 仓库简介

本仓库的主要用途是：

- 保存每周组会的 **PPT、PDF、Markdown、代码示例** 等资料  
- 方便组内成员 **查阅、复用、整理历史资料**  
- 建立长期的 **知识积累库**  

---

## 文件结构

仓库文件通常按 **日期** 或 **专题** 组织，例如：

按日期：

2026-03-30/
├─ presentation.pdf
├─ notes.md

按专题：


Machine_Learning/
├─ week1_notes.pdf
├─ week2_code_examples.ipynb


> 建议保持一致的命名规范，便于查找和管理。

---

## 使用说明

1. 浏览仓库查看历史分享资料  
2. 根据日期或专题查找你感兴趣的内容  
3. 可以自由下载或复用资料，但请注明出处  

---

## 上传新资料

每次组会结束后，上传资料的步骤：

1. 在仓库中新建一个文件夹，例如：

YYYY-MM-DD

或专题名称：

Topic_Name

2. 将会议相关文件上传到对应文件夹  
3. 提交（Commit）更新，并填写提交信息，例如：

Add 2026-03-30 weekly meeting notes


---

## 注意事项

- 尽量保持文件命名规范，方便查找  
- 避免上传过大的文件，可先压缩或上传到云盘并附链接  
- 仅上传与组会相关的资料，保持仓库整洁
